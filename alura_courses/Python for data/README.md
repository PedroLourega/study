# Python para Dados: Primeiros Passos

Repositório que contém os exercícios, atividades e desafios desenvolvidos durante o curso "Python para dados: Primeiros Passos" da Alura. O foco principal foi armazenar e consolidar os fundamentos da linguagem aplicados à Ciência de Dados.

## Conteúdo Desenvolvido

O curso presente aqui neste repositório abrange os seguintes temas:

* **Fundamentos da Linguagem:** Utilização de comandos básicos e sintaxe Python.
* **Tipagem de Dados:** Identificação e manipulação de tipos de variáveis e tipos básicos de dados.
* **Estruturas de Controle:** Implementação de estruturas condicionais e de repetição (loops).
* **Estruturas de Dados:** Trabalho prático com listas, dicionários e outras estruturas essenciais.
* **Aplicação Prática:** Desenvolvimento de lógica de programação voltada à construção de projetos de análise de dados.
